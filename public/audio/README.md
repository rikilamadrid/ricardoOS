# Aero FM audio

Drop one MP3 per track here, named `<track-id>.mp3`. The IDs come from
`src/data/music.ts`:

- `endless-summer-03.mp3`
- `mall-hours-2am.mp3`
- `poolside-telephone.mp3`
- `aero-glass-dreams.mp3`
- `vhs-memory-foam.mp3`

Use only royalty-free / properly licensed vaporwave (e.g. Pixabay Music,
Free Music Archive CC tracks, or your own). Files are served statically from
`/audio/<id>.mp3`. Until a file exists, that track shows as unavailable in the
player.
